\# 🏥 digitalHMS — Hospital Management System



A full-featured Hospital Management System built with \*\*Django 4.2\*\* and \*\*MySQL\*\*, 

designed to streamline hospital operations including patient management, 

doctor scheduling, pharmacy, billing, and lab reports.



\## 🚀 Features



\- 👤 Custom authentication for Admin, Doctor, and Staff

\- 🧑‍⚕️ Patient registration and appointment management

\- 💊 Pharmacy and lab report management

\- 🧾 Billing and prescription generation with print support

\- 🔒 Security hardened with session management and input validation

\- 🐛 20+ production-level bug fixes applied



\## 🛠️ Tech Stack



| Layer | Technology |

|-------|------------|

| Backend | Python, Django 4.2 |

| Database | MySQL |

| Frontend | HTML, CSS, JavaScript |

| Auth | Custom login system |



\## ⚙️ Setup Instructions



1\. Clone the repository

&#x20;  git clone https://github.com/Sreelal-A/digitalHMS.git



2\. Install dependencies

&#x20;  pip install -r requirements.txt



3\. Configure your MySQL database in settings.py



4\. Run migrations

&#x20;  python manage.py migrate



5\. Start the server

&#x20;  python manage.py runserver
